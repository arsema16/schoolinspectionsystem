# Quick Reference Guide
## School Longitudinal Inspection System

---

## Quick Start

### For Admins
1. **Login** → http://localhost:5000/login.html
2. **Import Data** → Admin Management → Import Students
3. **View Dashboard** → Automatic after login
4. **Generate Report** → Scroll to Report Generation section

### For Inspectors
1. **Login** → http://localhost:5000/login.html
2. **Apply Filters** → Select year, grade, subject
3. **Review Data** → Check statistics and charts
4. **Generate Report** → Select type and download

---

## Key Metrics Explained

| Metric | What It Means | Good Value |
|--------|---------------|------------|
| **Total Students** | Number of students in selected filters | N/A |
| **Average Performance** | Mean grade across all students | > 70% |
| **Pass Rate** | % of students with grades ≥ 50% | > 90% |
| **Red Flags** | Students at risk of failing | < 10% of total |

## Filter Combinations

### Common Scenarios

**Scenario 1: Check overall school performance**
- Years: All Years
- Grade: All Grades
- Subject: Overall
- Result: Complete school overview

**Scenario 2: Find struggling Math students in Grade 9**
- Years: 2017
- Grade: 9
- Subject: Maths
- Result: Grade 9 students failing Math

**Scenario 3: Compare boys vs girls performance**
- Run twice:
  1. Gender: Male
  2. Gender: Female
- Compare results

## Color Codes

- 🟢 **Green**: Good performance, positive trends
- 🔴 **Red**: Critical issues, immediate action needed
- 🟠 **Orange**: Warning, needs attention
- 🔵 **Blue**: Neutral, informational

- Modify any data

- View audit logs

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't login | Check username/password, contact admin |
| No data showing | Reset filters to "All" |
| Report failed | Wait and retry, check internet |
| Session expired | Normal after 8 hours, login again |
| Access denied | You may not have permission |
